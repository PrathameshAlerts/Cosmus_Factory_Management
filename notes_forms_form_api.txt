Django Form  API


from django import forms

class studentregistration(forms.Form):
    name = forms.CharField()
    email = forms.EmailField()

in html 
label for = id_name
id =  id_name
name = name
label element = Name

type = text or email depending upon char or email field

<p><label for="id_name">Name:</label> 
<input id="id_name" type="text" name="name" maxlength="100" required></p>

<p><label for="id_email">Email:</label> 
<input type="email" name="email" id="id_email" required></p>

form rendring options
{{form}} will render all
{{form.as_table}} will render with table tags
{{form.as_p}} will render with p tags
{{form.as_ul}} will render with ul
{{form.name}} or {{form.email}} render individual fields
